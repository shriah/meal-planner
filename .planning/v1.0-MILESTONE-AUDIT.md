## Tech Debt

### Phase 06: save-history-export (3 items)

1. **Dead font file** — `public/fonts/inter-regular.ttf` (26 KB) is committed but never loaded. Route handler loads `inter-regular.woff`. Either delete the `.ttf` or align the route path.

2. **Dead re-export** — `plan-store.ts` re-exports `addWeeks` from `week-utils`. `WeekNavigator.tsx` imports it directly from the source; the re-export has no consumer.

3. **Silent persistence failures** — All 6 store mutations call `saveWeekPlan()` fire-and-forget. IndexedDB quota exhaustion would cause silent plan divergence. Accepted trade-off per Phase 4 decision (keeps UI responsive); worth monitoring in production.
