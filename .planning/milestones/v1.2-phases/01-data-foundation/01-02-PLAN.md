Create `src/services/food-db.ts` with these exact exports:

```typescript
import { db } from '@/db/client';
import type { ComponentRecord } from '@/types/component';
import type { ComponentType, BaseType } from '@/types/component';
import type { MealRecord } from '@/types/meal';
import type { UserPreferencesRecord } from '@/types/preferences';
```

**Component CRUD:**

1. `getAllComponents(): Promise<ComponentRecord[]>` — returns `db.components.toArray()`

2. `getComponentsByType(type: ComponentType): Promise<ComponentRecord[]>` — returns `db.components.where('componentType').equals(type).toArray()`

3. `getExtrasByBaseType(baseType: BaseType): Promise<ComponentRecord[]>` — query `db.components.where('componentType').equals('extra').toArray()` then filter in JS: `extras.filter(e => (e.compatible_base_types ?? []).includes(baseType))`

4. `addComponent(component: Omit<ComponentRecord, 'id'>): Promise<number>` — sets `created_at` to `new Date().toISOString()` if empty string, then calls `db.components.add(...)` and returns the auto-generated ID

5. `updateComponent(id: number, changes: Partial<ComponentRecord>): Promise<void>` — calls `db.components.update(id, changes)`

6. `deleteComponent(id: number): Promise<void>` — calls `db.components.delete(id)`

**Meal CRUD:**

7. `getAllMeals(): Promise<MealRecord[]>` — returns `db.meals.toArray()`

8. `addMeal(meal: Omit<MealRecord, 'id'>, extraIds: number[]): Promise<number>` — wraps in `db.transaction('rw', db.meals, db.meal_extras, async () => { ... })`. Inside: add meal with `created_at` set, then `bulkAdd` meal_extras rows `{ meal_id: mealId, component_id: eid }` for each extraId. Return the meal ID.

9. `deleteMeal(id: number): Promise<void>` — wraps in `db.transaction('rw', db.meals, db.meal_extras, async () => { ... })`. Delete all meal_extras where `meal_id === id`, then delete the meal.

10. `getMealExtras(mealId: number): Promise<ComponentRecord[]>` — query `db.meal_extras.where('meal_id').equals(mealId).toArray()`, extract component_ids, then `db.components.bulkGet(ids)` and filter out undefined.

**Preferences:**

11. `getPreferences(): Promise<UserPreferencesRecord | undefined>` — returns `db.preferences.get('prefs')`

12. `putPreferences(prefs: UserPreferencesRecord): Promise<void>` — calls `db.preferences.put(prefs)` (upsert with fixed key 'prefs')
