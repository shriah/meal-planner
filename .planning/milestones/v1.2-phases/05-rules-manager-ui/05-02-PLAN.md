    Actions:
    ```typescript
    type FormAction =
      | { type: 'SET_NAME'; name: string }
      | { type: 'SET_RULE_TYPE'; ruleType: 'day-filter' | 'no-repeat' | 'require-component' }
    ```

    Reducer:
    - `SET_RULE_TYPE`: Reset ALL type-specific fields to defaults. Keep `name`. Defaults:
      - `day-filter`: `{ days: [], slots: [], filter: {} }`

      - `require-component`: `{ component_id: null, days: [], slots: [] }`
    - `SET_NAME`: Only update name, keep rest.
    - `SET_DAYS`, `SET_SLOTS`, `SET_FILTER`, `SET_COMPONENT_TYPE`, `SET_COMPONENT_ID`: Spread current state and update the specific field.
    - `LOAD_PRESET`: Replace entire state with provided state.

    Initial state: `{ name: '', ruleType: '' }`.

    ```typescript
    const EXAMPLE_PRESETS: Record<string, FormState> = {
    ```
