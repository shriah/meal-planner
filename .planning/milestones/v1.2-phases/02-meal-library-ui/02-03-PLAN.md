Render a grid/table with:

- Row headers: "Rice-based", "Bread-based", "Other"
- Each cell: shadcn `Checkbox` with `checked={baseTypeSlots[baseType]?.includes(slot) ?? false}` and `onCheckedChange={() => onToggle(baseType, slot)}`.

The `onToggle` handler in SlotSettings toggles the slot in/out of the base type's array:
```typescript
const handleToggle = (baseType: BaseType, slot: MealSlot) => {
  setBaseTypeSlots(prev => {
    const current = prev[baseType] ?? []
    const next = current.includes(slot)
      ? current.filter(s => s !== slot)
      : [...current, slot]
    return { ...prev, [baseType]: next }
  })
}
```

Grid uses Tailwind: `grid grid-cols-4 gap-4 items-center` with proper alignment.
